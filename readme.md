# RGR: Робота із версіонуванням бази даних (Liquibase)

## Опис
У цій роботі реалізовано керування версіями бази даних MongoDB за допомогою Liquibase.

## Використані технології
- Liquibase
- MongoDB
- Docker
- GitHub

## База даних
Назва: productsdb  
Колекція: products  

## Основні дії
- створення колекції
- додавання даних
- виконання update
- виконання rollback

## Команди
- liquibase status
- liquibase update
- liquibase rollbackCount 1
